A growing collection of small, self-contained scripts — organized by platform.

## Index

| Category | Description |
|----------|-------------|
| [`roblox/`](roblox/) | Roblox (Luau) client/server scripts |

_More categories will be added as the collection grows (e.g. `python/`, `shell/`, `js/`)._

## Structure

```
.
├── README.md          ← this index
└── roblox/            ← Roblox Luau scripts
    └── README.md      ← index of Roblox scripts
```

Each category folder has its own `README.md` listing the scripts inside it, what
they do, and how to use them. When adding a new script:

1. Drop it in the matching category folder (create one if it's a new platform).
2. Add a row for it in that folder's `README.md`.
3. Add the category here if it's new.
